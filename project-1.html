Project Overview

Shopper is an e-commerce platform designed for watch enthusiasts. Users can browse and purchase a wide range of watches and conveniently pay online. The admin panel offers comprehensive features including the ability to add new products, view and manage orders, update inventory levels, and oversee customer accounts,

Shopper is an e-commerce web application for selling watches online. It includes features for product management, user authentication, shopping cart functionality, and order management. here

E-commerce Watch Store Features
Product Management: Add, update, delete, and view watches through an intuitive admin interface. Effortlessly manage product listings and details.

User Authentication: Secure user registration and login with JWT-based authentication and session management. Role-based access ensures seamless operations for both admins and customers.

Shopping Cart & Checkout: Add products to your cart and complete the checkout process with ease. Integrates smoothly with payment gateways like Stripe and PayPal for a secure transaction experience.

Order Management: View and manage customer orders efficiently. Track order status from placement to delivery, ensuring a smooth customer experience.
